Your site uses **heavy dark gradient overlays** (60% black from left, 80% from bottom), so the images must:
- Have **rich detail and warm lighting** (twilight, golden hour, dusk — NOT midday flat light)
- Feature the **home as the clear focal point** (not landscapes with a distant house)
- Be **centered or slightly right** in frame (left side gets darkened most)
- Have **no people** visible
- Feel **cinematic, moody, premium** — not bright and cheerful

## Usage Notes
- Generate at **2560x1440** minimum for hero images (displayed full-screen)
- Generate a second version at **800x600** or crop for tile thumbnails
- All images will have heavy dark gradient overlays applied via CSS
- Warm lighting (golden/blue hour) is critical — midday flat lighting will look dead under the overlays
- Keep composition clean with the home as the clear focal point
- **After downloading from Unsplash**, run the conversion script below to auto-resize and convert to WebP

## Converting Downloaded Images
After downloading your picks, place them in `public/neighborhoods/source/` named by slug:
```
public/neighborhoods/source/newport-beach.jpg
public/neighborhoods/source/costa-mesa.jpg
... etc
```
Then run:
```bash
node scripts/convert-neighborhood-images.mjs
```
